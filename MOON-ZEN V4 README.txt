# Moon-Zen V4 - Zombie Attack Script

## Overview

**Moon-Zen V4** is a powerful and customizable script designed for the Roblox game **"Zombie Attack."** It enhances your gameplay with features such as Auto Aim, Wallhack/ESP, and extensive rainbow-themed customization options. Built using the **Rayfield GUI library**, Moon-Zen V4 offers a user-friendly interface and robust performance for an optimized gaming experience.

## Features

- **Auto Aim & Headshot:** Automatically lock onto and aim at the nearest zombieâ€™s head, providing precise headshots and a tactical advantage.

- **Wallhack/ESP:** See zombies through walls with customizable ESP. Includes a dynamic rainbow color mode for an enhanced visual experience.

- **Rainbow Customization:** Apply rainbow colors to ESP, Auto Aim indicators, and the entire GUI theme for a visually striking and personalized gameplay experience.

- **Error Handling & Compatibility:** Includes error handling with `pcall` and `xpcall` to ensure smooth operation on any executor, preventing crashes and ensuring reliability.

- **User-Friendly Interface:** Powered by the Rayfield GUI library, the script features an intuitive and clean design that makes configuration easy for both experienced and new users.
